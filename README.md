# The Intelligibility Monument Digital Platform Pro · Ultra-Realistic V5

This is the corrected V5 deployment package.

## V5 Fixes
- Rotation works more clearly through drag, slider, Rotate Left, Rotate Right, and Auto Rotate.
- The main monument appears larger in the 360 viewer.
- Clicking a hotspot updates the selected spot panel on the right with its QR code.
- The Selected Spot QR button still opens a large QR modal.
- Main runtime files are `styles-v5.css` and `app-v5.js`.

## Deployment
Upload these items to the root of your GitHub repository:
- index.html
- styles-v5.css
- app-v5.js
- manifest.webmanifest
- assets/

Then hard refresh the published site.
