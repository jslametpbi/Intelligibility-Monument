# The Intelligibility Monument · Final Fixed Build

This package fixes the 360 viewer and the selected-spot QR panel.

## Fixed
- Clean title: `Rotate the Monument 360°`
- Rotation controls: Rotate Left, Rotate Right, Auto Rotate, Reset, Slider, and Drag
- When a hotspot is clicked, its QR code appears in the right-side panel
- `Selected Spot QR` opens a larger QR modal
- Uses simple standard files: `index.html`, `styles.css`, and `app.js`
- This avoids the old app-v3/app-v4/app-v5 cache confusion

## Upload to GitHub Pages

Upload the contents inside this folder to the repository root:

- index.html
- styles.css
- app.js
- manifest.webmanifest
- assets/

After upload, hard refresh your GitHub Pages link:
- Mac: Cmd + Shift + R
